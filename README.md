# Monopoly Party Edition

Browser-based Monopoly-inspired game focused on **local multiplayer** (hot-seat mode) so friends and family can play together on one PC.

## Features

- 2 to 6 players.
- Turn-based dice rolling.
- Buy properties, pay rent, taxes, and card events.
- Railroads and utilities with dynamic rent.
- Bankruptcy elimination and winner detection.
- Modern, responsive UI for desktop/laptop play.

## Run

No build step required.

1. Open `index.html` in a browser, or run a quick static server:
   - `python3 -m http.server 8080`
2. Visit `http://localhost:8080`.
3. Configure players and start.
